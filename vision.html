"3.47 e+21" significa un 3 seguido de 21 ceros. El juego está preparado para cifras que ninguna calculadora normal maneja — y mostrarlas siempre claras.

La invitación dorada

"ASCENDER" aparece solo cuando vale la pena. Nunca presiona: es un premio que el jugador se ganó.

Claridad ante todo

El gancho visual · el juego entero evoluciona

Tres épocas, tres mundos distintos

A medida que el imperio crece, todo cambia: el fondo, los colores, la propia Forja. Esa transformación es la recompensa visual del progreso — y el material perfecto para los anuncios del juego.

ÉPOCA 1

El comienzo helado

Un dron solitario sobre un asteroide. Azules fríos, silencio, vacío. Aquí empieza la demo de hoy.

ÉPOCA 2

El fuego industrial

Refinerías, reactores, astilleros. El universo se ilumina con las máquinas que el jugador construyó. El calor se siente como poder.

ÉPOCA 3

Ingeniería de estrellas

Ya no se construyen fábricas: se rediseñan sistemas solares. Violetas, oro y una escala que impresiona.
