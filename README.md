# Stellar Forge — Playable Demo (Vertical Slice)

- **Play:** https://impalagrim21.github.io/stellar-forge-demo/
- **Full-game vision:** https://impalagrim21.github.io/stellar-forge-demo/vision.html

Compiled demo build only — the game source lives in a private repository.

This is the **vertical slice**: the full core loop running on the real,
server-ready economy engine (1485 unit + integration tests green). Tap the
Forge to earn Credits, buy and stack the nine generator tiers (Mining Drone →
Dyson Swarm), hit ×2 owned-count milestones, and Ascend to mint Dark Matter for
a permanent head start.

Built with TypeScript, Phaser 4 and Vite.
